# Centro de Servicio para Motocicletas y Venta de SOAT
¡Bienvenido a nuestro sitio web de reserva de citas y venta de SOAT. Este sitio ofrece una plataforma integral diseñada para simplificar y facilitar el proceso de programación de citas y la gestión de clientes en un centro de servicio para motocicletas. Además, brindamos la conveniencia de adquirir SOAT para todo tipo de vehículos.

## Características Destacadas
- Gestión Integral de Clientes
- Con nuestro sitio web, los usuarios tienen la capacidad de:

- Ver, registrar, editar y eliminar datos de clientes fácil y rápidamente.
Utilizar la funcionalidad CRUD (Crear, Leer, Actualizar, Eliminar) para gestionar eficientemente la base de datos de clientes, manteniendo un registro actualizado de cada cliente.

## Diseño Atractivo y Responsivo
Para el diseño y desarrollo del front-end del sitio web, hemos utilizado:
- Lenguajes como HTML, CSS y Bootstrap.
- Estas tecnologías permiten la creación de una interfaz atractiva y fácil de usar con un aspecto moderno y receptivo.
La combinación de HTML para la estructura, CSS para el estilo y Bootstrap para la implementación de componentes y estilos predefinidos asegura una experiencia visualmente agradable y consistente en todos los dispositivos y navegadores.

## Integración Efectiva Front-End y Back-End
El back-end del sitio web ha sido implementado utilizando:

El lenguaje de programación Python.
Python actúa como un puente entre el front-end y la base de datos MySQL.
Con Python (Flask), podemos manejar solicitudes y respuestas del cliente, así como realizar operaciones en la base de datos, como inserciones, actualizaciones y eliminaciones.
Esta integración perfecta entre el front-end y el back-end garantiza un flujo de datos eficiente y una experiencia de usuario fluida.

## Venta de SOAT para Todos los Vehículos
Además de la gestión de clientes y la reserva de citas, nuestro sitio web también ofrece:

- La venta de SOAT para todo tipo de vehículos.
- Los usuarios tienen la opción de acceder a información detallada sobre los diferentes tipos de SOAT disponibles y realizar compras directamente a través del sitio web.
- Esto brinda comodidad y fácil acceso para los clientes, evitando la necesidad de visitar físicamente el centro de servicio para adquirir SOAT.
